# SlackWebApi

Quick and dirty [Slack Web API](https://api.slack.com/web) wrapper. Ain't it grand?

## Installation

Soonly:

```elixir
def deps do
  [
    {:slack_web_api, "~> 0.1.0"}
  ]
end
```

And add your Slack bot token to your app config:

```elixir
config :slack_web_api, bot_token: YOUR_TOKEN
```
